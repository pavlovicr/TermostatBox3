\hypertarget{md_managed__components_2lvgl____lvgl_2demos_2README_autotoc_md406}{}\doxysubsection{\texorpdfstring{Music player}{Music player}}\label{md_managed__components_2lvgl____lvgl_2demos_2README_autotoc_md406}
The music player demo shows what kind of modern, smartphone-\/like user interfaces can be created on LVGL. It works the best with display with 480x272 or 272x480 resolution.

See in \href{https://github.com/lvgl/lvgl/tree/master/demos/music}{\texttt{ music}} folder.

\hypertarget{md_managed__components_2lvgl____lvgl_2demos_2README_autotoc_md407}{}\doxysubsection{\texorpdfstring{Keypad and encoder}{Keypad and encoder}}\label{md_managed__components_2lvgl____lvgl_2demos_2README_autotoc_md407}
LVGL allows you to control the widgets with a keypad and/or encoder without a touchpad. This demo shows how to handle buttons, drop-\/down lists, rollers, sliders, switches, and text inputs without touchpad. Learn more about the touchpad-\/less usage of LVGL \href{https://docs.lvgl.io/master/overview/indev.html\#keypad-and-encoder}{\texttt{ here}}.

See in \href{https://github.com/lvgl/lvgl/tree/master/demos/keypad_encoder}{\texttt{ keypad\+\_\+encoder}} folder.
