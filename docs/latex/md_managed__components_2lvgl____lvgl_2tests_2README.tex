\begin{DoxyItemize}
\item {\bfseries{Unit Tests}}\+: Standard functional tests in {\ttfamily src/test\+\_\+cases/} with screenshot comparison capabilities
\item {\bfseries{Performance Tests}}\+: ARM-\/emulated benchmarks in {\ttfamily src/test\+\_\+cases\+\_\+perf/} running on QEMU/\+SO3 environment
\item {\bfseries{Emulated Benchmarks}}\+: Automated {\ttfamily lv\+\_\+demo\+\_\+benchmark} runs in ARM emulation to prevent performance regressions
\end{DoxyItemize}

\item {\bfseries{Docker Testing}}\+: Build with {\ttfamily docker build . -\/f tests/\+Dockerfile -\/t lvgl\+\_\+test\+\_\+env} then run
\item {\bfseries{Performance Testing}}\+: Use {\ttfamily ./tests/perf.py test} (requires Docker + Linux)
\item {\bfseries{Benchmark Testing}}\+: Use {\ttfamily ./tests/benchmark\+\_\+emu.py run} for emulated performance benchmarks (requires Docker + Linux)
\end{DoxyItemize}

The performance tests are run inside a Docker container that launches an ARM emulated environment using QEMU to ensure consistent timing across machines. Each test runs on a lightweight ARM-\/based OS (\href{https://github.com/smartobjectoriented/so3}{\texttt{ SO3}}) within this emulated environment.

To run the tests\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{./perf.py\ [-\/-\/clean]\ [-\/-\/auto-\/clean]\ [-\/-\/test-\/suite\ <suite>]\ [-\/-\/build-\/options\ <option>]\ [build|generate|test]}

\end{DoxyCode}



\begin{DoxyItemize}
\item {\ttfamily build} and {\ttfamily generate}\+: generates all necessary build and configuration files
\item {\ttfamily test}\+: launches Docker with the appropriate volume mounts and runs the tests inside the container
\end{DoxyItemize}

\begin{quote}
\mbox{[}!\+NOTE\mbox{]} Building doesn\textquotesingle{}t actually build the source files because the current docker image doesn\textquotesingle{}t separate the building and running. Instead, it does both \end{quote}
You can specify different build configurations via {\ttfamily -\/-\/build-\/options}, and optionally filter tests using {\ttfamily -\/-\/test-\/suite}.

For full usage options, run\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{./perf.py\ -\/-\/help}

\end{DoxyCode}
