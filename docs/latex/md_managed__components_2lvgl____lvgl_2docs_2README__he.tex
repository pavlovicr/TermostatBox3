{\bfseries{LVGL}} היא ספריית UI חינמית וקוד פתוח שמאפשרת ליצור ממשקי משתמש גרפיים לכל MCU או MPU מכל ספק ועל כל פלטפורמה.

{\bfseries{דרישות}}\+: ל־\+LVGL אין תלות חיצונית, מה שמקל על הידור לכל יעד מודרני, החל מ־\+MCU קטנים ועד MPU מבוססי Linux מרובי ליבות עם תמיכת 3D. עבור UI פשוט דרושים רק \texorpdfstring{$\sim$}{\string~}100kB RAM, כ\texorpdfstring{$\sim$}{\string~}200–300kB Flash ומאגר רינדור בגודל 1/10 מגודל המסך.

{\bfseries{להתחלה מהירה}}\+: בחרו פרויקט מוכן ל־\+VSCode,‏ Eclipse או כל סביבת פיתוח אחרת ונסו את LVGL על המחשב. קוד ה־\+UI של LVGL בלתי תלוי בפלטפורמה, ולכן ניתן להשתמש באותו קוד גם ביעדים משובצים.

{\bfseries{LVGL Pro}} הוא מארז כלים מלא לבנייה, בדיקה, שיתוף והפצה מהירה יותר של UI. הוא כולל עורך XML ליצירה ולבדיקה מהירה של רכיבים לשימוש חוזר, ייצוא קוד C או טעינת קובצי XML בזמן ריצה. מידע נוסף באתר.\hypertarget{md_managed__components_2lvgl____lvgl_2docs_2README__he_autotoc_md434}{}\doxysubsection{\texorpdfstring{💡 יכולות}{💡 יכולות}}\label{md_managed__components_2lvgl____lvgl_2docs_2README__he_autotoc_md434}
{\bfseries{חינמית וניידת}}
\begin{DoxyItemize}
\item ספריית C מלאה (תואמת ++C) ללא תלות חיצונית.
\item ניתנת לקימפול לכל MCU או MPU, עם כל (RT)OS. תומכת ב־\+Make,‏ CMake וב־globbing פשוט.
\item תמיכה במסכי מונוכרום, e\+Paper,‏ OLED או TFT, ואף במסכים של מחשבים. ראו \href{https://docs.lvgl.io/master/details/main-modules/display/index.html}{\texttt{ תצוגות}}
\item רישיון MIT המקל על שימוש מסחרי.
\item נדרשים רק 32kB RAM ו־128kB Flash, מאגר מסגרת, ולפחות מאגר רינדור בגודל 1/10 מגודל המסך.
\item מערכת הפעלה, זיכרון חיצוני ו־\+GPU נתמכים אך אינם נדרשים.
