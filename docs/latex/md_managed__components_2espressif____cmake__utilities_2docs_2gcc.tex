\chapter{Link Time Optimization(LTO)}
\hypertarget{md_managed__components_2espressif____cmake__utilities_2docs_2gcc}{}\label{md_managed__components_2espressif____cmake__utilities_2docs_2gcc}\index{Link Time Optimization(LTO)@{Link Time Optimization(LTO)}}

 Link time optimization(\+LTO) improves the optimization effect of GCC, such as reducing binary size, increasing performance, and so on. For more details please refer to related \href{https://gcc.gnu.org/onlinedocs/gccint/LTO.html}{\texttt{ GCC documents}}.\hypertarget{md_managed__components_2espressif____cmake__utilities_2docs_2gcc_autotoc_md86}{}\doxysection{\texorpdfstring{Use}{Use}}\label{md_managed__components_2espressif____cmake__utilities_2docs_2gcc_autotoc_md86}
To use this feature, you need to include the required CMake file in your project\textquotesingle{}s CMake\+Lists.\+txt after {\ttfamily project(\+XXXX)}.


\begin{DoxyCode}{0}
\DoxyCodeLine{include(\$ENV\{IDF\_PATH\}/tools/cmake/project.cmake)}
\DoxyCodeLine{}
\DoxyCodeLine{project(XXXX)}
\DoxyCodeLine{}
\DoxyCodeLine{include(gcc)}

\end{DoxyCode}


The LTO feature is disabled by default. To use it, you should enable the option {\ttfamily CU\+\_\+\+GCC\+\_\+\+LTO\+\_\+\+ENABLE} in menuconfig. Then specify target components or dependencies to be optimized by LTO after {\ttfamily include(gcc)} as follows\+:

\DoxyCodeLine{}
\DoxyCodeLine{cu\_gcc\_string\_1byte\_align(COMPONENTS\ component\_c\ component\_d}
\DoxyCodeLine{\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ DEPENDS\ dependence\_c\ dependence\_d)}

\end{DoxyCode}


Based on your requirement, set compiling optimization level in the option {\ttfamily COMPILER\+\_\+\+OPTIMIZATION}.

It is recommended that users had better optimize application components and dependencies than kernel and hardware driver ones.\hypertarget{md_managed__components_2espressif____cmake__utilities_2docs_2gcc_autotoc_md88}{}\doxysection{\texorpdfstring{Example}{Example}}\label{md_managed__components_2espressif____cmake__utilities_2docs_2gcc_autotoc_md88}

Configure {\ttfamily ESP32-\/\+C2} as the target platform, enable {\ttfamily CU\+\_\+\+GCC\+\_\+\+LTO\+\_\+\+ENABLE} and {\ttfamily CONFIG\+\_\+\+COMPILER\+\_\+\+OPTIMIZATION\+\_\+\+ASSERTIONS\+\_\+\+DISABLE}, set {\ttfamily COMPILER\+\_\+\+OPTIMIZATION} to be {\ttfamily -\/Os}. Increase the {\ttfamily main} task stack size to {\ttfamily 5120} by option {\ttfamily ESP\+\_\+\+MAIN\+\_\+\+TASK\+\_\+\+STACK\+\_\+\+SIZE}.
