Lock LVGL\textquotesingle{}s general mutex. LVGL is not thread safe, so a mutex is used to avoid executing multiple LVGL functions at the same time from different threads. It shall be called when calling LVGL functions from threads different than lv\+\_\+timer\+\_\+handler\textquotesingle{}s thread. It doesn\textquotesingle{}t need to be called in LVGL events because they are called from \doxylink{lv__timer_8c_adb5cfb35b8a8ffe081b95c92b29c7c8c}{lv\+\_\+timer\+\_\+handler()}.
