C driver for Invensense ICM42607/\+ICM42670 6-\/axis gyroscope and accelerometer based on I2C communication.\hypertarget{md_managed__components_2espressif____icm42670_2README_autotoc_md389}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md_managed__components_2espressif____icm42670_2README_autotoc_md389}

\begin{DoxyItemize}
\item Get 3-\/axis accelerometer and 3-\/axis gyroscope data, either raw or as floating point values.
\item Read temperature from ICM42607/\+ICM42670 internal temperature sensor.
\item Configure gyroscope and accelerometer sensitivity.
\item ICM42607/\+ICM42670 power down mode.
\end{DoxyItemize}\hypertarget{md_managed__components_2espressif____icm42670_2README_autotoc_md390}{}\doxysection{\texorpdfstring{Limitations}{Limitations}}\label{md_managed__components_2espressif____icm42670_2README_autotoc_md390}
