Lock LVGL\textquotesingle{}s general mutex. LVGL is not thread safe, so a mutex is used to avoid executing multiple LVGL functions at the same time from different threads. It shall be called when calling LVGL functions from threads different than lv\+\_\+timer\+\_\+handler\textquotesingle{}s thread.
