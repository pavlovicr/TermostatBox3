\hypertarget{md_managed__components_2lvgl____lvgl_2docs_2README__jp_autotoc_md517}{}\doxysubsection{\texorpdfstring{📒 概要}{📒 概要}}\label{md_managed__components_2lvgl____lvgl_2docs_2README__jp_autotoc_md517}
{\bfseries{LVGL}} は、あらゆるメーカーやプラットフォームの MCU および MPU に対応した、無料でオープンソースの UI ライブラリです。

\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}要件\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}：\+LVGL は外部依存がないため、最新のあらゆるターゲットに簡単にコンパイルできます。小型 MCU から 3D 対応のマルチコア Linux MPU まで対応。シンプルな UI の場合、約 100kB の RAM、200～300kB のフラッシュ、および画面の 1/10 サイズのバッファで十分です。

\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}はじめに\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}：\+VSCode、\+Eclipse などのプロジェクトを選び、\+PC 上で LVGL を試してみましょう。\+LVGL の UI コードは完全にプラットフォーム非依存なので、そのまま組込みターゲットでも使用できます。

{\bfseries{LVGL Pro}} は、\+UI をより迅速に構築・テスト・共有・出荷できる完全なツールキットです。\+XML エディターを備え、再利用可能なコンポーネントの作成・テスト、C コードのエクスポート、\+XML の実行時読み込みが可能です。詳細は公式サイトをご覧ください。\hypertarget{md_managed__components_2lvgl____lvgl_2docs_2README__jp_autotoc_md518}{}\doxysubsection{\texorpdfstring{💡 機能}{💡 機能}}\label{md_managed__components_2lvgl____lvgl_2docs_2README__jp_autotoc_md518}
{\bfseries{無料でポータブル}}
\begin{DoxyItemize}
\item 外部依存のない完全な C（\+C++ 互換）ライブラリ。
\item あらゆる MCU / MPU、任意の (RT)OS に対応。\+Make、\+CMake、glob 構文をサポート。
\item モノクロ、電子ペーパー、\+OLED、\+TFT、さらにはモニターにも対応。\href{https://docs.lvgl.io/master/main-modules/display/index.html}{\texttt{ ディスプレイ}}
\item MIT ライセンスで商用利用も簡単。
\item 必要リソースはわずか 32kB RAM、128kB フラッシュ、フレームバッファ、および画面の 1/10 の描画バッファ。
\item OS、外部メモリ、\+GPU の使用は任意。
\end{DoxyItemize}

{\bfseries{ウィジェット・スタイル・レイアウトなど}}
\begin{DoxyItemize}
\item 30 以上の \href{https://docs.lvgl.io/master/widgets/index.html}{\texttt{ ウィジェット}}：ボタン、ラベル、スライダー、チャート、キーボード、メーター、アーク、テーブルなど。
\item 約 100 のプロパティを持つ柔軟な \href{https://docs.lvgl.io/master/common-widget-features/styles/index.html}{\texttt{ スタイルシステム}}。
\item \href{https://docs.lvgl.io/master/common-widget-features/layouts/flex.html}{\texttt{ Flexbox}}、\href{https://docs.lvgl.io/master/common-widget-features/layouts/grid.html}{\texttt{ Grid}} レイアウトエンジンによる自動配置。
\item UTF-\/8 文字レンダリング。\+CJK、タイ語、ヒンディー語、アラビア語、ペルシャ語などをサポート。
\item \href{https://docs.lvgl.io/master/main-modules/observer/index.html}{\texttt{ データバインディング}} により、\+UI とアプリケーションを簡単に接続。
\item レンダリングエンジンはアニメーション、アンチエイリアス、不透明度、スムーズスクロール、影、画像変換をサポート。
\item \href{https://docs.lvgl.io/master/libs/gltf.html}{\texttt{ 3D レンダリングエンジン}} により \href{https://sketchfab.com/}{\texttt{ gl\+TF モデル}} を Open\+GL で表示可能。
\item マウス、タッチパッド、キー入力、外部ボタン、エンコーダなどに対応。\href{https://docs.lvgl.io/master/main-modules/indev.html}{\texttt{ 入力デバイス}}
\item \href{https://docs.lvgl.io/master/main-modules/display/overview.html\#how-many-displays-can-lvgl-use}{\texttt{ マルチディスプレイ}} をサポート。
\end{DoxyItemize}\hypertarget{md_managed__components_2lvgl____lvgl_2docs_2README__jp_autotoc_md519}{}\doxysubsection{\texorpdfstring{📦️ 対応プラットフォーム}{📦️ 対応プラットフォーム}}\label{md_managed__components_2lvgl____lvgl_2docs_2README__jp_autotoc_md519}

LVGL Pro は、組込み UI を効率的に構築・テスト・共有・出荷するための完全なツール群です。

以下の 4 つのツールで構成されています：


\begin{DoxyEnumerate}
\item \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+XML エディター\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}：\+LVGL Pro の中心。\+XML でコンポーネントや画面を作成し、データバインディング、翻訳、アニメーション、テストなどを管理。詳細は \href{https://docs.lvgl.io/master/xml/xml/index.html}{\texttt{ XML フォーマット}} および \href{https://docs.lvgl.io/master/xml/editor/index.html}{\texttt{ エディター}}。
\item \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}オンラインビューア\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}：ブラウザでエディターを実行し、\+Git\+Hub プロジェクトを開いて簡単に共有。環境構築は不要。アクセス：\href{https://viewer.lvgl.io}{\texttt{ https\+://viewer.\+lvgl.\+io}}
\item \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+CLI ツール\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}：C コードの生成や CI/\+CD テストを自動化。詳細は \href{https://docs.lvgl.io/master/xml/tools/cli.html}{\texttt{ こちら}}。
\item \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+Figma プラグイン\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}：\+Figma から直接スタイルを同期・抽出。\href{https://docs.lvgl.io/master/xml/tools/figma.html}{\texttt{ 詳細はこちら}}。
\end{DoxyEnumerate}

これらのツールを組み合わせることで、開発者は効率的に UI を構築し、信頼性の高いテストとチームコラボレーションを実現できます。

詳細は \href{https://pro.lvgl.io}{\texttt{ https\+://pro.\+lvgl.\+io}} へ。\hypertarget{md_managed__components_2lvgl____lvgl_2docs_2README__jp_autotoc_md521}{}\doxysubsection{\texorpdfstring{🤝 商用サービス}{🤝 商用サービス}}\label{md_managed__components_2lvgl____lvgl_2docs_2README__jp_autotoc_md521}

\begin{DoxyItemize}
\item \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}グラフィックデザイン\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}：社内デザイナーが、製品とハードウェア性能に最適化された美しくモダンなデザインを提供。
\item \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+UI 実装\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}：お客様または当社が作成したデザインに基づき、最適な UI を実装。必要なウィジェットがない場合も新たに開発します。
\item \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}コンサルティング・サポート\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}：\+UI 開発での時間やコストの無駄を避けるための専門支援。
\item \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}ボード認証\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}：開発ボードや製品キットを提供する企業向けに、\+LVGL の動作実績を示す認証を提供。
\end{DoxyItemize}
