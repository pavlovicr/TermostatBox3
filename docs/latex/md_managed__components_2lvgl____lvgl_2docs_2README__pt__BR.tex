{\bfseries{LVGL}} é uma biblioteca de UI gratuita e de código aberto que permite criar interfaces gráficas para qualquer MCU e MPU, de qualquer fabricante, em qualquer plataforma.

{\bfseries{Requisitos}}\+: o LVGL não possui dependências externas, o que facilita compilar para qualquer alvo moderno, desde pequenos MCUs até MPUs Linux multicore com suporte a 3D. Para uma UI simples, você precisa de \texorpdfstring{$\sim$}{\string~}100 kB de RAM, \texorpdfstring{$\sim$}{\string~}200–300 kB de flash e um buffer de renderização com 1/10 do tamanho da tela.

{\bfseries{Para começar}}, escolha um projeto pronto para uso no VSCode, Eclipse ou outro, e experimente o LVGL no seu PC. O código de UI do LVGL é totalmente independente de plataforma, então você pode reutilizá-\/lo nos seus alvos embarcados.

{\bfseries{LVGL Pro}} é um kit completo para ajudar você a criar, testar, compartilhar e entregar UIs com mais rapidez. Ele inclui um Editor XML onde você pode criar e testar componentes reutilizáveis, exportar código C ou carregar os XMLs em tempo de execução. Saiba mais aqui.\hypertarget{md_managed__components_2lvgl____lvgl_2docs_2README__pt__BR_autotoc_md528}{}\doxysubsection{\texorpdfstring{💡 Recursos}{💡 Recursos}}\label{md_managed__components_2lvgl____lvgl_2docs_2README__pt__BR_autotoc_md528}
{\bfseries{Gratuito e portátil}}
\begin{DoxyItemize}
\item Biblioteca totalmente em C (compatível com C++) sem dependências externas.
\item Pode ser compilada para qualquer MCU ou MPU, com qualquer (RT)OS. Make, CMake e globbing simples são suportados.
\item Suporta displays monocromáticos, e\+Paper, OLED, TFT e até monitores. \href{https://docs.lvgl.io/master/main-modules/display/index.html}{\texttt{ Displays}}
\item Licença MIT, o que facilita o uso em projetos comerciais.
\item Precisa de apenas 32 kB de RAM, 128 kB de Flash, um frame buffer e pelo menos um buffer de 1/10 da tela para renderização.
\item Sistema operacional, memória externa e GPU são suportados, mas não obrigatórios.
\end{DoxyItemize}

{\bfseries{Widgets, estilos, layouts e mais}}
\begin{DoxyItemize}
\item 30+ \href{https://docs.lvgl.io/master/widgets/index.html}{\texttt{ Widgets}} embutidos\+: Button, Label, Slider, Chart, Keyboard, Meter, Arc, Table e muitos outros.
\item \href{https://docs.lvgl.io/master/common-widget-features/styles/index.html}{\texttt{ Sistema de estilos}} flexível com \texorpdfstring{$\sim$}{\string~}100 propriedades para personalizar qualquer parte dos widgets em qualquer estado.
\item Motores de layout tipo \href{https://docs.lvgl.io/master/common-widget-features/layouts/flex.html}{\texttt{ Flexbox}} e \href{https://docs.lvgl.io/master/common-widget-features/layouts/grid.html}{\texttt{ Grid}} para dimensionar e posicionar widgets de forma responsiva.
\item Texto renderizado com codificação UTF-\/8, com suporte a CJK, tailandês, híndi, árabe e persa.
\item \href{https://docs.lvgl.io/master/main-modules/observer/index.html}{\texttt{ Data bindings}} para conectar facilmente a UI ao aplicativo.
\item Motor de renderização com suporte a animações, antialiasing, opacidade, rolagem suave, sombras, transformação de imagens e mais.
\item \href{https://docs.lvgl.io/master/libs/gltf.html}{\texttt{ Poderoso motor de renderização 3D}} para exibir \href{https://sketchfab.com/}{\texttt{ modelos gl\+TF}} com Open\+GL.
\item Suporte a mouse, touchpad, keypad, teclado, botões externos e encoder. \href{https://docs.lvgl.io/master/main-modules/indev.html}{\texttt{ Dispositivos de entrada}}
\item Suporte a \href{https://docs.lvgl.io/master/main-modules/display/overview.html\#how-many-displays-can-lvgl-use}{\texttt{ múltiplos displays}}.
\end{DoxyItemize}\hypertarget{md_managed__components_2lvgl____lvgl_2docs_2README__pt__BR_autotoc_md529}{}\doxysubsection{\texorpdfstring{📦️ Plataformas compatíveis}{📦️ Plataformas compatíveis}}\label{md_managed__components_2lvgl____lvgl_2docs_2README__pt__BR_autotoc_md529}
