\doxysubsection{Podroben opis}
The module provides C bindings for the Thor\+VG library. Please refer to src/examples/\+Capi.\+cpp to find the thorvg\+\_\+capi usage examples. 

The thorvg\+\_\+capi module allows to implement the Thor\+VG client and provides the following functionalities\+:
\begin{DoxyItemize}
\item drawing shapes\+: line, curve, polygon, circle, user-\/defined, ...
\item filling\+: solid, linear and radial gradient
\item scene graph \& affine transformation (translation, rotation, scale, ...)
\item stroking\+: width, join, cap, dash
\item composition\+: blending, masking, path clipping
\item pictures\+: SVG, PNG, JPG, bitmap 
\end{DoxyItemize}
