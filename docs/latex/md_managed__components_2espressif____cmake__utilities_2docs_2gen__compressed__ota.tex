 When using the compressed OTA, we need to generate the compressed app firmware. This document mainly describes how to generate the compressed app firmware.

For more information about compressed OTA, refer to \href{https://github.com/espressif/esp-iot-solution/tree/master/components/bootloader_support_plus}{\texttt{ bootloader\+\_\+support\+\_\+plus}}.\hypertarget{md_managed__components_2espressif____cmake__utilities_2docs_2gen__compressed__ota_autotoc_md90}{}\doxysection{\texorpdfstring{Use}{Use}}\label{md_managed__components_2espressif____cmake__utilities_2docs_2gen__compressed__ota_autotoc_md90}
In order to use this feature, you need to include the needed CMake file in your project\textquotesingle{}s CMake\+Lists.\+txt after {\ttfamily project(\+XXXX)}.


\begin{DoxyCode}{0}
\DoxyCodeLine{project(XXXX)}
\DoxyCodeLine{}
\DoxyCodeLine{include(gen\_compressed\_ota)}

\end{DoxyCode}


Generate the compressed app firmware in an ESP-\/\+IDF "{}project"{} directory by running\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{idf.py\ gen\_compressed\_ota}

\end{DoxyCode}


This command will compile your project first, then it will generate the compressed app firmware. For example, run the command under the project {\ttfamily simple\+\_\+ota\+\_\+examples} folder. If there are no errors, the {\ttfamily custom\+\_\+ota\+\_\+binaries} folder will be created and contains the following files\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{simple\_ota.bin.xz\ \ }
\DoxyCodeLine{simple\_ota.bin.xz.packed}

\end{DoxyCode}


The file named {\ttfamily simple\+\_\+ota.\+bin.\+xz.\+packed} is the actual compressed app binary file to be transferred.

In addition, if \href{https://docs.espressif.com/projects/esp-idf/en/latest/esp32c3/security/secure-boot-v2.html}{\texttt{ secure boot}} is enabled, the command will generate the signed compressed app binary file\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{simple\_ota.bin.xz.packed.signed}

\end{DoxyCode}


you can also use the script \href{https://github.com/espressif/esp-iot-solution/tree/master/tools/cmake_utilities/scripts/gen_custom_ota.py}{\texttt{ gen\+\_\+custom\+\_\+ota.\+py}} to compress the specified app\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{python3\ gen\_custom\_ota.py\ -\/i\ simple\_ota.bin}

\end{DoxyCode}
