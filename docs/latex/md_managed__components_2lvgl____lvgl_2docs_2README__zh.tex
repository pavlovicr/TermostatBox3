\hypertarget{md_managed__components_2lvgl____lvgl_2docs_2README__zh_autotoc_md465}{}\doxysubsection{\texorpdfstring{📒 概述}{📒 概述}}\label{md_managed__components_2lvgl____lvgl_2docs_2README__zh_autotoc_md465}
{\bfseries{LVGL}} 是一个免费、开源的 UI 库，可用于为来自任何厂商、任何平台的 MCU 或 MPU 创建图形用户界面。

\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}要求\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}：\+LVGL 无需外部依赖，因此可以轻松编译到任何现代平台上，从小型 MCU 到支持 3D 的多核 Linux MPU。简单 UI 仅需约 100kB RAM、200–300kB Flash，以及 1/10 屏幕大小的渲染缓冲区。

\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}开始使用\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}：选择一个可直接使用的 VSCode、\+Eclipse 或其他项目，在 PC 上尝试 LVGL。\+LVGL 的 UI 代码完全平台无关，因此也可直接用于嵌入式设备。

{\bfseries{LVGL Pro}} 是一套完整的工具包，可让你更快地构建、测试、共享和发布 UI。它附带一个 XML 编辑器，你可以在其中快速创建和测试可重用组件，导出 C 代码，或在运行时加载 XML。了解更多信息请点击这里。\hypertarget{md_managed__components_2lvgl____lvgl_2docs_2README__zh_autotoc_md466}{}\doxysubsection{\texorpdfstring{💡 特性}{💡 特性}}\label{md_managed__components_2lvgl____lvgl_2docs_2README__zh_autotoc_md466}
{\bfseries{免费与可移植性}}
\begin{DoxyItemize}
\item 纯 C（兼容 C++）库，无外部依赖。
\item 可为任意 MCU 或 MPU 编译，支持任何 (RT)OS。支持 Make、\+CMake 和简单文件匹配。
\item 支持单色、电子墨水、\+OLED、\+TFT 显示器，甚至是 PC 显示器。详见 \href{https://docs.lvgl.io/master/details/main-modules/display/index.html}{\texttt{ 显示模块}}
\item 采用 MIT 许可证，可自由用于商业项目。
\item 仅需 32kB RAM、128kB Flash、一个帧缓冲区，以及至少 1/10 屏幕大小的渲染缓冲。
\item 支持但不强制要求使用操作系统、外部存储或 GPU。
\end{DoxyItemize}

{\bfseries{控件、样式、布局等}}
\begin{DoxyItemize}
\item 超过 30 个内置 \href{https://docs.lvgl.io/master/details/widgets/index.html}{\texttt{ 控件}}：按钮、标签、滑块、图表、键盘、仪表、弧形、表格等。
\item 灵活的 \href{https://docs.lvgl.io/master/details/common-widget-features/styles/index.html}{\texttt{ 样式系统}}，提供约 100 种属性，可自定义控件任意部分与状态。
\item \href{https://docs.lvgl.io/master/details/common-widget-features/layouts/flex.html}{\texttt{ Flexbox}} 和 \href{https://docs.lvgl.io/master/details/common-widget-features/layouts/grid.html}{\texttt{ Grid}} 布局引擎可实现自适应布局。
\item 文本支持 UTF-\/8 编码，兼容中日韩、泰语、印地语、阿拉伯语、波斯语等。
\item \href{https://docs.lvgl.io/master/details/auxiliary-modules/observer/index.html}{\texttt{ 数据绑定}} 模块轻松连接 UI 与应用逻辑。
\item 渲染引擎支持动画、抗锯齿、不透明度、平滑滚动、阴影、图像变换等。
\item \href{https://docs.lvgl.io/master/details/libs/gltf.html}{\texttt{ 强大的 3D 渲染引擎}}，可使用 Open\+GL 显示 \href{https://sketchfab.com/}{\texttt{ gl\+TF 模型}}。
\item 支持鼠标、触摸板、按键、键盘、外部按钮、编码器等 \href{https://docs.lvgl.io/master/details/main-modules/indev.html}{\texttt{ 输入设备}}。
\item 支持 \href{https://docs.lvgl.io/master/details/main-modules/display/overview.html\#how-many-displays-can-lvgl-use}{\texttt{ 多显示屏}}。
\end{DoxyItemize}\hypertarget{md_managed__components_2lvgl____lvgl_2docs_2README__zh_autotoc_md467}{}\doxysubsection{\texorpdfstring{📦️ 平台支持}{📦️ 平台支持}}\label{md_managed__components_2lvgl____lvgl_2docs_2README__zh_autotoc_md467}
