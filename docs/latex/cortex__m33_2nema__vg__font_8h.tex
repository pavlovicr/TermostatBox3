\doxysubsection{Podroben opis}
Vector font rendering. 

This file includes the necessary structs and functions that are used for rendering text (strings and single characters), using vector fonts. The accompanying vector font converter utility, converts truetype fonts (ttf files) to instances of the structs defined here. A use case of this module is included in the respective examples (examples/\+Nema\+VG/render\+\_\+vg\+\_\+font).

Creates an 8-\/bpp raster version of the bound vector font. Performs dynamic memory allocation in the graphics memory (for the font bitmaps) and in the heap (for the data structs accessed by the CPU). When the font is no longer needed, function "{}nema\+\_\+vg\+\_\+destroy\+\_\+raster\+\_\+font()"{} can be used to free the allocated memory. The font generation may fail when there is not enough memory to generate the font or when the font size is greater than the height of the framebuffer.
