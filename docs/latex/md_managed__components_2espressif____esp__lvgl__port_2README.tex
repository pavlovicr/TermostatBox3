\hypertarget{md_managed__components_2espressif____esp__lvgl__port_2README_autotoc_md371}{}\doxysubsection{\texorpdfstring{Detecting gestures}{Detecting gestures}}\label{md_managed__components_2espressif____esp__lvgl__port_2README_autotoc_md371}
LVGL (version 9.\+4 and higher) includes support for software detection of multi-\/touch gestures. This detection can be enabled by setting the {\ttfamily LV\+\_\+\+USE\+\_\+\+GESTURE\+\_\+\+RECOGNITION} config and having {\ttfamily ESP\+\_\+\+LCD\+\_\+\+TOUCH\+\_\+\+MAX\+\_\+\+POINTS} \texorpdfstring{$>$}{>} 1. Example usage of the gesture callback can be found in LVGL \href{https://docs.lvgl.io/master/details/main-modules/indev/gestures.html}{\texttt{ documentation}}.

The LVGL port task is responsible for passing the touch coordinates to the gesture recognizers and calling the registered callback when a gesture is detected.

To correctly distinguish two finger swipe from rotation, we recommend changing the default value (which is 0) for the rotation threshold. From our testing we recommend starting with 0.\+15 radians.

\hypertarget{md_managed__components_2espressif____esp__lvgl__port_2README_autotoc_md374}{}\doxysection{\texorpdfstring{Power Saving}{Power Saving}}\label{md_managed__components_2espressif____esp__lvgl__port_2README_autotoc_md374}
The LVGL port can be optimized for power saving mode. There are two main features.\hypertarget{md_managed__components_2espressif____esp__lvgl__port_2README_autotoc_md375}{}\doxysubsection{\texorpdfstring{LVGL task sleep}{LVGL task sleep}}\label{md_managed__components_2espressif____esp__lvgl__port_2README_autotoc_md375}
For optimization power saving, the LVGL task should sleep, when it does nothing. Set {\ttfamily task\+\_\+max\+\_\+sleep\+\_\+ms} to big value, the LVGL task will wait for events only.

The LVGL task can sleep till these situations\+:
\begin{DoxyItemize}
\item LVGL display invalidate
\item LVGL animation in process
\item Touch interrupt
\item Button interrupt
\item Knob interrupt
\item USB mouse/keyboard interrupt
\item Timeout ({\ttfamily task\+\_\+max\+\_\+sleep\+\_\+ms} in configuration structure)
\item User wake (by function {\ttfamily lvgl\+\_\+port\+\_\+task\+\_\+wake})
\end{DoxyItemize}

\begin{quote}
\mbox{[}!\+WARNING\mbox{]} This feature is available from LVGL 9. \end{quote}
\begin{quote}
\mbox{[}!\+NOTE\mbox{]} Don\textquotesingle{}t forget to set the interrupt pin in LCD touch when you set a big time for sleep in {\ttfamily task\+\_\+max\+\_\+sleep\+\_\+ms}. \end{quote}
