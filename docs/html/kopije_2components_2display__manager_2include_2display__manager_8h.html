Display Manager for ESP-BOX-3 with LVGL.

Ta komponenta inicializira in upravlja zaslon ESP-BOX-3. Poskrbi za:

  • Inicializacijo LVGL sistema
  • Nastavitev ESP-BOX-3 driverjev
  • Kreiranje osnovnega UI kontejnerja
