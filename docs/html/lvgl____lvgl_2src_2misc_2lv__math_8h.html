Calculate the atan2 of a vector.

Parametri
x
y
Rezultat(i)
the angle in degree calculated from the given parameters in range of [0..360]

Fast XY vector to integer degree algorithm - Jan 2011 www.RomanBlack.com Converts any XY values including 0 to a degree value that should be within +/- 1 degree of the accurate value without needing large slow trig functions like ArcTan() or ArcCos(). NOTE! at least one of the X or Y values must be non-zero! This is the full version, for all 4 quadrants and will generate the angle in integer degrees from 0-360. Any values of X and Y are usable including negative values provided they are between -1456 and 1456 so the 16bit multiply does not overflow.
