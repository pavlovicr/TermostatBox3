Podroben opis

Codec audio sample information Notes: channel_mask is used to filter wanted channels in driver side when set to 0, default filter all channels when channel is 2, can filter channel 0 (set to 1) or channel 1 (set to 2) when channel is 4, can filter either 3,2 channels or 1 channel.
