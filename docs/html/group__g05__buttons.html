◆ bsp_iot_button_create()

esp_err_t bsp_iot_button_create ( button_handle_t  btn_array[],
int *  btn_cnt,
int  btn_array_size 
)

Initialize all buttons.

Returned button handlers must be used with espressif/button component API

Opomba
For LCD panel button which is defined as BSP_BUTTON_MAIN, bsp_display_start should be called before call this function.
Parametri
[out]btn_arrayOutput button array
[out]btn_cntNumber of button handlers saved to btn_array, can be NULL
[in]btn_array_sizeSize of output button array. Must be at least BSP_BUTTON_NUM
Rezultat(i)
  • ESP_OK All buttons initialized
  • ESP_ERR_INVALID_ARG btn_array is too small or NULL
  • ESP_FAIL Underlaying iot_button_create failed
