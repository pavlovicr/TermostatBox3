Podroben opis

Button matrix key configuration. Just need to configure the GPIO associated with this GPIO in the matrix keyboard.

Matrix Keyboard Layout (3x3):

| Button 1 | Button 2 | Button 3 | | (R1-C1) | (R1-C2) | (R1-C3) | |-----------------------------------—| | Button 4 | Button 5 | Button 6 | | (R2-C1) | (R2-C2) | (R2-C3) | |-----------------------------------—| | Button 7 | Button 8 | Button 9 |

| (R3-C1) | (R3-C2) | (R3-C3) |

  • Button matrix key is driven using row scanning.
  • Buttons within the same column cannot be detected simultaneously, but buttons within the same row can be detected without conflicts.
