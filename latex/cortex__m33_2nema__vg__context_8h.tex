\doxysubsection{Podroben opis}
Nema\+VG Context interface. 

Contains Nema\+VG error codes, fill rules, rendering quality defines and functions for updating various rendering parameters. The functions defined here can be used to access the context parameters. The Context is an internal (opaque) struct of Nema\+VG.
