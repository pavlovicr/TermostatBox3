There are cases where it is not appropriate to present the contents of a set of subdirectories in alphabetical order. When this is the case, a pseudo-\/directive in the {\ttfamily index.\+rst} file in the parent directory can be specified to govern the sequence its subdirectories are processed. The example below is from {\ttfamily lvgl/examples/widgets/index.\+rst}. It is provided in order to cause the "{}\+Base Widget"{} (obj) directory to be processed first (and thus included in the output first).
