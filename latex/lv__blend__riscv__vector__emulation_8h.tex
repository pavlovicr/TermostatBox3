\doxysubsection{Podroben opis}
Software emulation of RISC-\/V Vector Extension (RVV 1.\+0) intrinsics

This header provides pure C implementations of RVV intrinsics to enable testing and verification on non-\/\+RVV platforms. The implementations follow the RVV specification as documented in\+: \href{https://dzaima.github.io/intrinsics-viewer/}{\texttt{ https\+://dzaima.\+github.\+io/intrinsics-\/viewer/}}

Usage\+:
\begin{DoxyEnumerate}
\item On systems without RVV support, include this header BEFORE \texorpdfstring{$<$}{<}riscv\+\_\+vector.\+h\texorpdfstring{$>$}{>}
\item Or define RISCV\+\_\+\+VECTOR\+\_\+\+EMULATION before including actual \texorpdfstring{$<$}{<}riscv\+\_\+vector.\+h\texorpdfstring{$>$}{>}
\item All \+\_\+\+\_\+riscv\+\_\+\texorpdfstring{$\ast$}{*} functions will be emulated in software
\end{DoxyEnumerate}

Limitations\+:
\begin{DoxyItemize}
\item No performance optimization (this is software emulation)
\item Vector length (vl) is tracked but all operations work on single elements in a loop
\item Predication and masking are simplified but functionally correct
\item LMUL \texorpdfstring{$<$}{<} 1 (fractional multipliers) are not supported 
\end{DoxyItemize}
