The music player demo shows what kind of modern, smartphone-\/like user interfaces can be created on LVGL. It works the best with display with 480x272 or 272x480 resolution.

\end{DoxyItemize}\hypertarget{md_managed__components_2lvgl____lvgl_2demos_2music_2README_autotoc_md466}{}\doxysection{\texorpdfstring{How the spectrum animation works}{How the spectrum animation works}}\label{md_managed__components_2lvgl____lvgl_2demos_2music_2README_autotoc_md466}

\begin{DoxyItemize}
\item {\ttfamily assets/spectrum.\+py} creates an array of spectrum values from a music. 4 band are created with 33 samples/sec\+: bass, bass-\/mid, mid, mid-\/treble.
\item The spectrum meter UI does the following\+:
\begin{DoxyItemize}
\item Zoom the album cover proportionality to the current bass value
\item Display the 4 bands on the left side of a circle by default at 0°, 45°, 90°, 135°
\item Add extra bars next to the "{}main bars"{} with a cosine shape. Add more bars for the lower bands.
\item If there is a large enough bass, add a random offset to the position of the bars. E.\+g. start from 63° instead of 0°. (bars greater than 180° start again from 0°)
\item If there is no bass, add 1 to the offset of the bars (it creates a "{}walking"{} effect)
\item Mirror the bars to the right side of the circle
\end{DoxyItemize}
