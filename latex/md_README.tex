Projekt implementira {\bfseries{samostojni Wi\+Fi termostat}} z grafičnim vmesnikom na dotik. Naprava\+:


\begin{DoxyItemize}
\item meri temperaturo in vlažnost prek {\bfseries{vgrajenega ali zunanjega senzorja}} (npr. SHT30, DHT22),
\item prikazuje trenutne vrednosti in nastavitve na {\bfseries{barvnem IPS zaslonu}} ESP32-\/\+S3-\/\+Box-\/3,
\item omogoča nastavljanje {\bfseries{ciljne temperature}} neposredno prek zaslona na dotik,
\item prek {\bfseries{HTTP REST API klicev}} krmili Shelly 2PM Gen3 — vklopi ali izklopi izhod (kanal 1) glede na izmereno temperaturo,
\item implementira {\bfseries{histerezno logiko}} za preprečitev pogostega preklapljanja,
\item opcijsko se integrira z {\bfseries{Home Assistant}} prek MQTT ali HTTP.
\end{DoxyItemize}
