Create new display panel. 

For maximum flexibility, this function performs only reset and initialization of the display. You must turn on the display explicitly by calling esp\+\_\+lcd\+\_\+panel\+\_\+disp\+\_\+on\+\_\+off(). The display\textquotesingle{}s backlight is not turned on either. You can use bsp\+\_\+display\+\_\+backlight\+\_\+on/off(), \doxylink{group__g04__display_ga12c7b53be5b337029f2d1aacce916d1c}{bsp\+\_\+display\+\_\+brightness\+\_\+set()} (on supported boards) or implement your own backlight control.

If you want to free resources allocated by this function, you can use esp\+\_\+lcd API, ie.\+:
