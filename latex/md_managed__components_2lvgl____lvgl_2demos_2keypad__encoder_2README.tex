LVGL allows you to control the widgets with keypad and/or encoder without touchpad. This demo shows how to handle buttons, drop-\/down lists, rollers, sliders, switches and text inputs without touchpad. Learn more about the touchpad-\/less usage of LVGL \href{https://docs.lvgl.io/master/overview/indev.html\#keypad-and-encoder}{\texttt{ here}}.
